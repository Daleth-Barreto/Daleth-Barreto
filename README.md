## 🔬 What I Somehow Work On

### Quantum Error Correction (QEC)
I try to make QEC decoders:
- faster than MWPM,
- simple enough to fit on real hardware,
- and honest about their limitations.

Current obsession:
**ConvSNN and GCN-SNN decoders** for Surface Codes (d = 3–7).

Pipeline I didn’t expect to learn:
**Stim → ML / SNN → RTL → FPGA**

No papers yet. Yes, I’m working on it.
